# API Contracts

OpenAPI specs, AsyncAPI specs, and interface contracts. Created by `/architecture`.

Typische Dateien (werden bei Bedarf erstellt):
- `api-v1.yaml` — REST API spec (OpenAPI 3.1)
- `events.yaml` — Event/message contracts (AsyncAPI 3.x)
- `PERF-BUDGETS.md` — Performance budgets (LCP, INP, CLS, query latencies)
- `AUDIO-BUDGETS.md` — Audio budgets (file sizes, voice limits, memory limits)
